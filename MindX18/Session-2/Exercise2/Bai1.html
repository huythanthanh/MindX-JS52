EXPORT

NAME EXPORT: Trong JavaScript ES6, named export được sử dụng để xuất nhiều thứ từ một module bằng cách thêm keyword export vào khai báo của chúng. Những thứ được export sẽ được phân biệt bằng tên. Sau đó import những thứ chúng ta cần sử dụng bằng cách bao quanh chúng cặp dấu ngoặc nhọn { }. Tên của module đã nhập phải giống với tên của module đã xuất.


DEFAULT EXPORT: Default Export trong Javascript ES6 chỉ cho phép xuất một mặc định cho mỗi file. Default Export có thể cho một function, class hoặc một object.


IMPORT

Để import một module vào một script thì bạn có thể sử dụng import


Cú pháp ở đây khá đơn giản, nó dạng như thế này: import * as name from "module-name"
